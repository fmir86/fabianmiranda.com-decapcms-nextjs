## 1. Executive Summary

The site has solid technical SEO foundations (schema markup, OG tags, canonical URLs, robots.txt), but is generating almost zero organic traffic. In 90 days: **7 clicks, 350 impressions, 2.0% CTR, avg position 16.2**. The site is essentially invisible for the target commercial keywords.

**Critical Finding:** The site ranks for almost nothing related to its target services. Only 5 unique queries triggered impressions in 90 days. The biggest opportunity -- "nearshore technology costa rica" (107 impressions) -- sits at position 35.6 with 0 clicks, and the blog post targeting it isn't even indexed by Google.

**Severity: HIGH** -- This site needs a fundamental content and SEO strategy overhaul to compete for the target keywords.

### 7.1 AI Bot Access
- robots.txt allows GPTBot, ClaudeBot, PerplexityBot, Google-Extended, Cohere, Applebot
- This is positive for being cited in AI-generated answers

### 7.2 GEO Readiness
| Factor | Status | Notes |
|--------|--------|-------|
| AI bot crawl access | Good | Explicitly allowed in robots.txt |
| Structured data | Good | Comprehensive schema markup helps AI understand content |
| Content depth | Weak | Blog posts exist but are few; need more authoritative, factual content |
| E-E-A-T signals | Weak | No testimonials, certifications, awards, publications, speaking engagements |
| Citability | Weak | Content lacks unique data points, statistics, frameworks that AI would cite |
| Entity establishment | Medium | Person schema exists but no Wikipedia, Crunchbase, or other knowledge graph entries |

### 7.3 What AI Engines Need to Cite You
For generative engines (ChatGPT, Perplexity, Google AI Overviews) to reference your site, you need:
- **Unique data or frameworks** (e.g., "The Miranda Nearshore ROI Calculator", original research)
- **Strong entity signals** (LinkedIn, Crunchbase, conference talks, podcast appearances)
- **Topical authority** -- More than 5 blog posts on your core topics
- **External citations** -- Other sites linking to your content as a reference

## 8. Recommendations (Priority Order)

### P0 -- Fix Immediately (This Week)

1. **Fix /services page 5xx error** -- This is your money page and Google can't even crawl it. Debug the server error and ensure it renders correctly.

2. **Fix /contact soft 404** -- Add sufficient content to the contact page so Google doesn't treat it as empty. Include a proper heading, description, and form.

3. **Get the nearshore blog post indexed** -- It's been crawled but not indexed. Options:
   - Improve content quality and uniqueness
   - Add internal links from homepage and services page
   - Request re-indexing via Search Console
   - Add author bio, publish date, and trust signals

4. **Rebuild sitemap** -- Run `npm run build` to regenerate sitemap.xml with all current pages, then resubmit to Search Console.

### P1 -- High Priority (This Month)

5. **Create dedicated landing pages for each target keyword:**
   - `/services/nearshore-web-development` -- Dedicated page with case studies, team info, timezone advantages, tech stack
   - `/services/ai-consultancy` -- AI strategy, assessment frameworks, implementation roadmap
   - `/services/ai-solutions-architecture` -- Architecture patterns, integration approaches, case studies
   - `/services/ai-implementation` -- Implementation methodology, timeline, deliverables

6. **Add "AI Solutions Architect" and "AI Implementation Consultant" to your content:**
   - Update homepage hero, about page bio, services page
   - Add to Person schema `jobTitle` and `knowsAbout`
   - Include in meta descriptions and title tags

7. **Add FAQ schema to services page** -- Convert existing FAQ content to JSON-LD FAQ schema for rich snippet eligibility.

8. **Write 2-3 new blog posts targeting keyword clusters:**
   - "How to Choose a Nearshore Web Development Partner in 2026"
   - "AI Implementation Roadmap: From POC to Production"
   - "Why Costa Rica is the Best Nearshore Destination for US Companies"

### P2 -- Medium Priority (Next 2-3 Months)

9. **Build topical authority:**
   - Publish 2-4 blog posts per month on nearshore development and AI implementation
   - Each post should target a specific long-tail keyword
   - Include original data, screenshots, code examples

10. **Improve E-E-A-T signals:**
    - Add client testimonials with company names
    - Display certifications (AWS, GCP, Azure AI)
    - Add a "Featured In" or "Speaking" section
    - Link to LinkedIn, GitHub with activity proof

11. **Build backlinks:**
    - Guest post on nearshore/outsourcing blogs
    - Get listed on Clutch.co, GoodFirms, TopDevelopers
    - Contribute to open-source projects and get profile links
    - Seek podcast/webinar appearances

12. **GEO optimization:**
    - Create "quotable" content blocks (statistics, definitions, frameworks)
    - Publish original research (e.g., survey on AI adoption in LATAM)
    - Ensure all content has clear, factual statements that AI can extract

### P3 -- Long-term (3-6 Months)

13. **Consider Spanish content** for LATAM market with hreflang tags
14. **Build a tools/resources section** (e.g., nearshore cost calculator, AI readiness assessment)
